# simple-vue-webpack

#### 一个简单的基于vue框架的webpack风格项目模板

----------

### 作者：草梅友仁

##  使用方法

```bash
npm i #下载依赖
npm run dev  #开发模式
npm run build  #生产模式，生成的文件在dist目录下
npm run server  #启用http服务器，可通过http://127.0.0.1:80 来访问
npm run lint:fix   #使用eslint修正风格
npm run uninstall  #移除所有node_modules
```
